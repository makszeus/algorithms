# Data Structures
A data structure is a set of data grouped in some way. 
For each data structure is defined:

- how data is stored in computer memory,
- what basic operations can be performed on this data and for what time.

Examples of data structures:
- Array
- Linked list
- Binary heap (it's a specialized tree–like data structure)

## Abstract Data Type
For an abstract type, an interface is defined — a set of operations that can be performed. An abstract type user, using these operations, can work with data without going into the internal details of the information storage mechanism.</br>
<br>If the algorithm works with data exclusively through the interface, then it will continue to function if one implementation of the interface is replaced with another. This is the essence of abstraction: the implementation is hidden behind the interface.</br>
<br>Implementations of abstract data types are concrete data structures. The implementation determines exactly how data is represented in memory and how this or that operation functions.</br>

Examples:
- List
- Stack
- Queue
- Deque
- Set
- Associative Array, Map, Dictionary
- Priority Queue
